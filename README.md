# Warehouse Management System (WMS)

## Overview

The Warehouse Management System (WMS) is a web-based application designed to manage inventory, track stock movements, and optimize warehouse operations. It provides features for managing products, orders, and users efficiently.

---

## Features

- User authentication
- Product inventory tracking
- Stock in/out management
- Order management

---

## Tech Stack

- **Frontend:** Node.js, TypeScript, NestJS
- **Backend:** ReactJS, NextJS
- **Database:** PostgreSQL
- **Authentication:** JWT-based authentication
